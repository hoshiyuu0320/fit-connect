# FIT-CONNECT プロジェクト概要

## 1. プロジェクト概要

FIT-CONNECTは、パーソナルトレーナー向けの顧客管理プラットフォームです。トレーナーが担当クライアントの情報・記録・コミュニケーションを一元管理できる環境を提供します。

### 2つの柱

| アプリ | 技術 | 状態 |
|--------|------|------|
| トレーナー向けWebアプリ | Next.js 15 (App Router) | 実装済み |
| 顧客向けモバイルアプリ | Flutter | 計画中 |

両アプリは共通バックエンドとして **Supabase**（PostgreSQL + Realtime + Storage + Auth）を使用します。

### 技術スタック概要

- **フレームワーク:** Next.js 15 (App Router)
- **バックエンド:** Supabase
- **状態管理:** Zustand
- **UI:** Tailwind CSS + Radix UI
- **フォーム:** React Hook Form + Zod

## 2. 実装済み機能一覧

### 認証

- ログイン `/login`、サインアップ `/signup`
- Supabase Auth による認証
- サインアップ時にトレーナーレコードを自動作成

### ダッシュボード `/dashboard`

- KPI統計カード（担当顧客数・今週メッセージ件数・アクティブ顧客数・要対応チケット数）
- 本日の予定（セッション一覧）
- 最近のメッセージプレビュー（最大5件）
- アラートリスト（非アクティブ顧客・期限切れ間近チケット）
- クイックアクション

### 顧客管理 `/clients`

- 顧客一覧（カード表示）
- 名前検索・フィルター（性別・年齢層・目的）
- QRコードによるクライアント招待
- 顧客詳細 `/clients/[client_id]`（6タブ構成）
  - **サマリー:** 体重・食事・運動・チケット概要
  - **体重:** 体重推移グラフ・テーブル
  - **食事:** 食事記録一覧（ページネーション対応）
  - **運動:** 運動記録一覧（ページネーション対応）
  - **カルテ:** トレーナーノート（作成・編集・削除・ファイル添付・顧客共有）
  - **チケット:** 発行済みチケット一覧
- プロフィール編集

### メッセージング `/message`

- リアルタイム送受信（Supabase Realtime）
- 画像添付（最大3枚、Supabase Storage）
- メッセージ編集（送信後5分以内）
- 返信（リプライ）機能
- 画像拡大表示

### スケジュール管理 `/schedule`

- カレンダー表示（日・週・月ビュー切り替え）
- セッション登録・編集・削除
- セッション検索
- カレンダーグリッドクリックで新規追加

### レポート・分析 `/report`

- クライアント別・期間別フィルター
- 統計カード（体重変化・食事件数・運動件数・合計運動時間）
- 体重推移グラフ
- 食事統計・運動統計
- CSV / PDF エクスポート

### チケット管理 `/tickets`

3タブ構成：

- **テンプレート:** チケットテンプレートのCRUD
- **チケット発行:** テンプレートからクライアントへ発行・発行済み一覧
- **月契約:** 月次自動更新契約の管理（active / paused / cancelled）

### 設定 `/settings`

- プロフィール編集（名前・アイコン画像）
- 対応スケジュール設定（曜日・時間帯）
- 通知設定
- メールアドレス変更 `/settings/email`
- パスワード変更 `/settings/password`
- ログアウト

## 4. データベース構成

### 主要テーブル

| テーブル名 | 用途 |
|-----------|------|
| `trainers` | トレーナー基本情報 |
| `clients` | 顧客情報（名前・性別・年齢・身長・目標体重・目的等） |
| `messages` | 双方向メッセージ（画像URL・編集フラグ・返信先ID） |
| `weight_records` | 体重記録 |
| `meal_records` | 食事記録（食事区分・カロリー・画像） |
| `exercise_records` | 運動記録（種目・時間・距離・カロリー） |
| `tickets` | 発行済みセッションチケット |
| `ticket_templates` | チケットテンプレート |
| `ticket_subscriptions` | 月契約（自動更新） |
| `sessions` | セッション予約 |
| `client_notes` | カルテ（トレーナーノート・添付ファイル・共有フラグ） |
| `trainer_schedules` | トレーナー対応スケジュール（曜日・時間帯） |

### Storageバケット

| バケット名 | 用途 |
|-----------|------|
| `client-notes` | カルテ添付ファイル |
| `message-images` | メッセージ画像 |

全テーブルはRow Level Security (RLS)で保護されています。詳細なスキーマ定義は `CLAUDE.md` および `src/types/` を参照してください。
