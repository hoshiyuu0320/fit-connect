## 顧客リスト画面

### パス
`/clients`

### 画面構成

#### 1. ヘッダーセクション
```
┌───────────────────────────────────────────────┐
│  顧客管理                           [検索 🔍]  │
└───────────────────────────────────────────────┘
```

**要素:**
- タイトル: "顧客管理"
- 検索ボックス: リアルタイム検索（プレースホルダー: "名前で検索..."）

#### 2. フィルターセクション
```
┌───────────────────────────────────────────────┐
│ フィルター: [すべて ▼] [すべて ▼] [すべて ▼] │
│            性別       年齢層     目的        │
└───────────────────────────────────────────────┘
```

**要素:**
- 性別フィルター: すべて / 男性 / 女性 / その他
- 年齢層フィルター: すべて / 10代 / 20代 / 30代 / 40代 / 50代 / 60代以上
- 目的フィルター: すべて / ダイエット / コンテスト / ボディメイク / 健康維持 / メンタル / パフォーマンス向上

**UI:**
- ドロップダウン or タグ選択式
- 選択中のフィルターはハイライト表示

#### 3. 顧客カードリスト

##### デスクトップ表示（グリッドレイアウト）
```
┌───────────────────┐ ┌───────────────────┐ ┌───────────────────┐
│ [性別色帯]        │ │ [性別色帯]        │ │ [性別色帯]        │
│  ┌────┐          │ │  ┌────┐          │ │  ┌────┐          │
│  │画像│ 山田太郎  │ │  │YK │ 佐藤花子  │ │  │TM │ 田中誠   │
│  └────┘ 28歳     │ │  └────┘ 35歳     │ │  └────┘ 42歳     │
│                   │ │                   │ │                   │
│  [詳細] [メッセージ]│ │  [詳細] [メッセージ]│ │  [詳細] [メッセージ]│
└───────────────────┘ └───────────────────┘ └───────────────────┘
```

##### モバイル表示（リストレイアウト）
```
┌─────────────────────────────────────┐
│ [青色帯] ┌────┐                     │
│          │画像│ 山田太郎 (28歳)     │
│          └────┘                     │
│          [詳細] [メッセージ]        │
├─────────────────────────────────────┤
│ [ピンク] ┌────┐                     │
│          │YK │ 佐藤花子 (35歳)     │
│          └────┘                     │
│          [詳細] [メッセージ]        │
└─────────────────────────────────────┘
```

**カード要素:**
- 性別色帯: カード左側に3px幅の色帯（男性=青、女性=ピンク、その他=黄色）
- プロフィール画像 or イニシャルアイコン（64x64px）
- 名前（太字、18px）
- 年齢（グレー、14px）
- 詳細ボタン（オプションA採用により、カード全体クリック可能のため実質不要だが、視覚的に配置）
- メッセージボタン（アイコン＋テキスト、クリックでメッセージ画面へ遷移）

**インタラクション:**
- カード全体: ホバーで背景色変化、クリックで詳細画面へ遷移
- メッセージボタン: クリック時は詳細画面遷移を防ぐ（イベント伝播停止）

#### 4. 空の状態（顧客が0件の場合）

## 顧客詳細画面

### パス
`/clients/[client_id]`

### 画面構成

#### 1. ページヘッダー
```
┌─────────────────────────────────────────────────────┐
│ ← 顧客リストに戻る                      [メッセージ] │
└─────────────────────────────────────────────────────┘
```

**要素:**
- 戻るリンク: `/clients`へ遷移
- メッセージボタン: `/message?clientId=xxx`へ遷移

#### 2. 基本情報セクション
```
┌─────────────────────────────────────────────────────┐
│  ┌──────┐                                           │
│  │      │  山田 太郎 (28歳) 男性                    │
│  │ 画像 │  職業: ITエンジニア                        │
│  │      │  身長: 175cm                              │
│  └──────┘  目標体重: 70kg                           │
│             現在の体重: 78kg (-2kg)                  │
│             目的: ボディメイク                       │
│                                                     │
│  目標: 「半年後のフィットネスコンテストで優勝!」    │
│       （トレーナーメモ: 筋力UPと体脂肪率減少が課題）│
│                                          [編集 ✏️]   │
└─────────────────────────────────────────────────────┘
```

**要素:**
- プロフィール画像（128x128px、円形）
- 名前（太字、24px）、年齢、性別
- 職業
- 身長
- 目標体重
- 現在の体重（最新の体重記録から取得、開始時からの変化量を表示）
- 目的（バッジ表示）
- 目標の詳細説明（トレーナーが記入）
- 編集ボタン（目標体重と目標説明を編集可能）※将来実装

#### 3. 体重推移グラフセクション
```
┌─────────────────────────────────────────────────────┐
│  体重推移                                           │
│                                                     │
│  kg                                                 │
│  85 ┐                                               │
│  80 ┤  ●                                            │
│  75 ┤    ●─●                                        │
│  70 ┤- - - - - - - - - - (目標体重)                 │
│  65 └───────────────────────                        │
│      1/1   1/8   1/15  1/22  日付                   │
│                                                     │
│  [1週間] [1ヶ月] [3ヶ月] [全期間]                   │
└─────────────────────────────────────────────────────┘
```

**要素:**
- タイトル: "体重推移"
- 折れ線グラフ
  - X軸: 日付
  - Y軸: 体重（kg）
  - 目標体重を破線で表示
  - データポイントにホバーでツールチップ表示（日付、体重）
- 期間選択ボタン（将来実装）

**使用ライブラリ候補:**
- Recharts
- Chart.js
- D3.js

## 共通UIコンポーネント

### 1. プロフィールアイコン（画像未設定時）
```
┌────┐
│ YT │  ← イニシャル（白文字、太字）
└────┘
  ↑
性別の色背景（円形）
```

**仕様:**
- サイズ: 64x64px（リスト）、128x128px（詳細）
- 形状: 円形
- 背景色: 性別に応じた色
  - 男性: #3B82F6（青）
  - 女性: #EC4899（ピンク）
  - その他: #F59E0B（黄色）
- テキスト: 名前の最大2文字のイニシャル、白色、太字

### 2. 性別バッジ
```
┌─────┐
│ 男性 │  ← 背景色: 青系
└─────┘

┌─────┐
│ 女性 │  ← 背景色: ピンク系
└─────┘

┌─────┐
│その他│  ← 背景色: 黄色系
└─────┘
```

### 3. 目的バッジ
```
┌──────────┐
│ ダイエット │  ← 背景色: 緑系
└──────────┘

┌──────────┐
│ コンテスト │  ← 背景色: 赤系
└──────────┘

┌────────────┐
│ ボディメイク │  ← 背景色: 青系
└────────────┘
```

### 4. ボタンスタイル

**プライマリボタン:**
- 背景色: #3B82F6（青）
- テキスト色: 白
- ホバー: #2563EB
- 例: メッセージボタン、詳細ボタン

**セカンダリボタン:**
- 背景色: 白
- テキスト色: #3B82F6
- ボーダー: 1px solid #3B82F6
- ホバー: 背景色 #EFF6FF
- 例: 戻るボタン、フィルタークリア

**アイコンボタン:**
- 背景なし
- テキスト色: #6B7280（グレー）
- ホバー: #3B82F6
- 例: 編集ボタン

## レスポンシブ対応

### ブレークポイント
- モバイル: 〜767px
- タブレット: 768px〜1023px
- デスクトップ: 1024px〜

### デスクトップ（1024px〜）
- 顧客リスト: 3カラムグリッド
- サイドバー: 常時表示（240px幅）
- 詳細画面: 2カラムレイアウト（基本情報 + グラフ/記録）

### タブレット（768px〜1023px）
- 顧客リスト: 2カラムグリッド
- サイドバー: 折りたたみ可能
- 詳細画面: 1カラムレイアウト

### モバイル（〜767px）
- 顧客リスト: 1カラムリスト
- サイドバー: ハンバーガーメニュー
- 詳細画面: 1カラム縦スクロール
- フィルター: ドロップダウンをボトムシートに変更

## アクセシビリティ

### キーボード操作
- Tab: フォーカス移動
- Enter: 選択/実行
- Escape: モーダル閉じる

### スクリーンリーダー対応
- 適切なARIAラベル設定
- 画像にalt属性
- フォーム要素にラベル関連付け

### 色覚多様性対応
- 性別の色分けに加えてアイコンも併用
- テキストと背景のコントラスト比 4.5:1以上
- 色だけに依存しない情報提示
